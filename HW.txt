What is async and defer?
"async" allows the script to be loaded and executed asynchronously, while "defer" allows the script to be loaded asynchronously but executed in order, just before the DOMContentLoaded event.


library(React, and JQuery) and framework(Angular, Vue ,Django ,Spring Boot ,Laravel ,Ruby on Rails) difference

A library is a collection of reusable code modules or functions that provide specific functionality, while a framework is a more comprehensive toolset that guides the overall application structure and provides a set of conventions to streamline development. 
Libraries offer more flexibility and allow developers to have more control, while frameworks provide a structured approach and take care of common tasks, reducing the need for manual implementation.

What is CDN? Why do we use it
A content delivery network (CDN) is a network of interconnected servers that speeds up webpage loading for data-heavy applications. CDN can stand for content delivery network or content distribution network.
 A CDN helps improve the page load time of your site and make it faster. 


What is crossorigin in script tag
The crossorigin attribute sets the mode of the request to an HTTP CORS Request. Web pages often make requests to load resources on other servers. 
The purpose of crossorigin attribute is used to share the resources from one domain to another domain.
